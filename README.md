# Weather App

A modern and responsive **Weather App** built with **React**, **TanStack Query**, **Tailwind CSS**, and **Shadcn UI**. This app allows users to view current weather data, forecasts, and manage favorite locations with an intuitive interface.

---

## Features

- **Current Location Weather**: Displays weather information for your current location, including:
  - Temperature
  - Feels like temperature
  - Sunrise & Sunset times
  - Wind speed and direction
- **5-Day Forecast**: Shows a 5-day weather forecast for selected cities.
- **City Search**: Search for any city and view its weather updates.
- **Favorites**: Add cities to your favorites for quick access.
- **Recent Searches**: Track and access your recent city searches.
- **Interactive Graph**: Compare current temperature vs. “feels like” temperature.
- **Custom Hooks**: Efficiently fetch and manage weather data using React custom hooks.
- **Responsive Design**: Fully responsive UI using Tailwind CSS and Shadcn UI components.

---

## Tech Stack

- **Frontend**: React.js + TypeScript
- **State & Data Fetching**: TanStack Query (React Query)
- **Styling**: Tailwind CSS, Shadcn UI
- **Data Visualization**: Chart for temperature comparison
- **API**: OpenWeatherMap (or other weather API)
